# Projeto: Preservação do Meio Ambiente

Este projeto foi desenvolvido utilizando as linguagens **HTML, CSS e JavaScript** com o objetivo de conscientizar os usuários sobre a importância da preservação do meio ambiente de forma simples e interativa.

A página apresenta informações sobre ações que ajudam a proteger a natureza, como economizar água, reciclar materiais, plantar árvores, reduzir o consumo de energia elétrica e utilizar meios de transporte menos poluentes. Essas atitudes contribuem para a conservação dos recursos naturais e para a melhoria da qualidade de vida.

O **HTML** foi utilizado para criar a estrutura da página, organizando títulos, textos, listas e botões. O **CSS** foi responsável pela aparência do site, aplicando cores relacionadas à natureza, espaçamento, alinhamento e estilos que tornam a página mais agradável e fácil de utilizar. Já o **JavaScript** adiciona interatividade ao projeto, permitindo exibir dicas ambientais aleatórias e atualizar um contador que simula o número de árvores preservadas, tornando a experiência mais dinâmica.

O principal objetivo deste projeto é demonstrar, de maneira prática, como as tecnologias de desenvolvimento web podem ser utilizadas para criar aplicações educativas. Além de reforçar conceitos de programação, o projeto incentiva a conscientização ambiental e mostra que pequenas ações do dia a dia podem gerar impactos positivos para o planeta.

Assim, o projeto une aprendizado em programação com educação ambiental, mostrando que a tecnologia também pode ser uma ferramenta importante para promover a sustentabilidade e incentivar hábitos mais responsáveis em relação ao meio ambiente.
